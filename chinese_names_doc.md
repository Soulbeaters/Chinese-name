# Документация системы обработки китайских имен

## 1. Особенности китайских имен

### 1.1 Структурные особенности
- Китайские имена обычно состоят из фамилии и имени
- Фамилия ставится перед именем (в отличие от западных имен)
- Фамилия обычно состоит из одного иероглифа (например, Ли, Ван, Чжан), но встречаются и двусложные фамилии (например, Оуян, Сыма, Чжугэ)
- Имя обычно состоит из 1-2 иероглифов

### 1.2 Отсутствие понятия отчества
- В традиционных китайских именах нет понятия "отчества", как в западных именах
- В современных китайских именах могут быть сложные фамилии или сложные имена, но структура остается двоичной: "фамилия + имя"
- При обработке данных мы устанавливаем поле "отчество" китайского имени как пустую строку

### 1.3 Системы транслитерации
Основные системы транслитерации китайских имен включают:
- **Пиньинь**: наиболее широко используемая система транслитерации китайского языка для английского и других западных языков
  - Пример: 李明 → Li Ming
- **Система Палладия**: система транслитерации китайского языка для русского языка
  - Пример: 李明 → Ли Мин
- **Система Уэйда-Джайлза**: историческая система транслитерации для английского языка
  - Пример: 李明 → Li Ming (с диакритическими знаками)
- **Йельская система**: система транслитерации, разработанная в Йельском университете
  - Пример: 李明 → Li Ming (с отличиями от пиньиня в обозначении тонов)

### 1.4 Региональные различия
Китайские имена могут различаться в зависимости от региона:
- **Материковый Китай**: упрощенные иероглифы, обычно используется пиньинь
- **Тайвань**: традиционные иероглифы, может использоваться система Уэйда-Джайлза или пиньинь
- **Гонконг, Макао**: традиционные иероглифы, может использоваться кантонская транслитерация или другие системы
- **Зарубежные китайцы**: могут использовать различные системы транслитерации в зависимости от страны проживания
- **Этнические меньшинства**: могут иметь специфические формы имен (тибетские, уйгурские, монгольские и др.)

## 2. Алгоритмы обработки китайских имен

### 2.1 Распознавание китайских иероглифов
- Используются диапазоны Unicode для распознавания китайских иероглифов:
  - Базовые иероглифы: U+4E00-U+9FFF
  - Расширение A: U+3400-U+4DBF
  - Совместимые идеограммы: U+F900-U+FAFF

### 2.2 Основные алгоритмы разделения имен
1. **Чистые китайские имена**:
   - Сначала проверяется, является ли фамилия двусложной (по предопределенному списку двусложных фамилий)
   - Если фамилия двусложная, первые 2-3 иероглифа считаются фамилией, остальные - именем
   - Если фамилия односложная, первый иероглиф считается фамилией, остальные - именем

2. **Транслитерированные китайские имена**:
   - Проверяется, присутствует ли фамилия в предопределенном списке транслитерированных фамилий
   - Определяется порядок имен (западный или китайский)
   - На основе положения фамилии определяется имя

### 2.3 Новые алгоритмы и возможности

#### 2.3.1 Обработка множественных систем транслитерации
- Поддержка пиньиня через библиотеку `pypinyin`
- Полная реализация системы Палладия с учетом особенностей русской фонетики
- Поддержка системы Уэйда-Джайлза и Йельской системы транслитерации
- Функция сравнения транслитераций для одного и того же имени в разных системах

#### 2.3.2 Обработка региональных различий
- Автоматическое распознавание и конвертация между упрощенными и традиционными иероглифами
- Специализированные правила для имен этнических меньшинств (тибетских, уйгурских, монгольских)
- Обработка региональных особенностей написания имен

#### 2.3.3 Расширенная обработка краевых случаев
- Обработка редких фамилий, отсутствующих в основной базе данных
- Поддержка смешанных имен, содержащих как китайские иероглифы, так и латинские буквы
- Распознавание и обработка японских имен с китайскими иероглифами
- Алгоритмы нечеткого поиска для обработки вариантов написания и ошибок

#### 2.3.4 Интеграция с системами управления данными
- Проверка корректности обработки китайских имен в существующей базе данных
- Интерфейс для взаимодействия с моделью ArticleAuthorship
- Функции пакетной обработки для эффективной обработки больших объемов данных
- Механизм ручной коррекции для особых случаев

## 3. База данных китайских имен

### 3.1 Структура базы данных фамилий
Расширенная база данных китайских фамилий включает:
- Более 500 распространенных китайских фамилий (односложных и двусложных)
- Для каждой фамилии хранится:
  - Написание иероглифами
  - Транслитерация в системе пиньинь
  - Транслитерация в системе Палладия
  - Частота использования
  - Региональное распространение

### 3.2 База данных китайских имен (личных)
Создана новая база данных китайских личных имен (данных при рождении):
- Часто используемые имена с указанием гендерной принадлежности
- Значение каждого имени
- Транслитерация в различных системах

### 3.3 Обратное отображение имен
- Поддержка определения китайских иероглифов по транслитерации
- Механизмы обработки неоднозначностей (один и тот же пиньинь может соответствовать разным иероглифам)
- Алгоритмы нечеткого сопоставления для поиска наиболее вероятных соответствий

## 4. Использование библиотек транслитерации

### 4.1 Библиотека pypinyin
- Используется для точного преобразования китайских иероглифов в пиньинь
- Поддерживает различные стили вывода (с тонами, без тонов, с инициалями и финалями и т.д.)
- Обеспечивает высокую точность преобразования для редких иероглифов
- **Интеграция и настройка**:
  - Библиотека интегрирована с возможностью автоматического определения гетерографических омонимов
  - Реализовано кэширование для повышения производительности при обработке больших объемов данных
  - Система автоматически деградирует до внутреннего словаря при отсутствии библиотеки

### 4.2 Конверсия между различными системами транслитерации
- Реализованы функции для преобразования между различными системами транслитерации
- Полный набор правил для системы Палладия позволяет точно транслитерировать имена для русскоязычных источников
- Поддержка других систем (Уэйда-Джайлза, Йельской) для международной совместимости
- **Новые возможности сравнения**:
  - Функция `compare_transliterations` позволяет сравнивать одно и то же имя в разных системах транслитерации
  - Поддерживается оценка фонетического сходства между различными транслитерациями
  - Реализован алгоритм обратного преобразования транслитерации в иероглифы

## 5. Тестирование и примеры использования

### 5.1 Примеры обработки китайских имен
```python
# Чистые китайские имена
split_chinese_name("李明")  # -> ('李', '明', '')
split_chinese_name("欧阳修")  # -> ('欧阳', '修', '')

# Транслитерированные имена (пиньинь)
identify_transliterated_chinese_name("Li Ming")  # -> ('李', '明', '')
identify_transliterated_chinese_name("Ming Li")  # -> ('李', '明', '')

# Транслитерированные имена (система Палладия)
identify_transliterated_chinese_name("Ли Мин")  # -> ('李', '明', '')
identify_transliterated_chinese_name("Оуян Сю")  # -> ('欧阳', '修', '')

# Преобразование между системами транслитерации
transliterate_chinese_name("李明", system="pinyin")  # -> "Li Ming"
transliterate_chinese_name("李明", system="palladius")  # -> "Ли Мин"
transliterate_chinese_name("李明", system="wade_giles")  # -> "Li Ming"
```

### 5.2 Обработка особых случаев
```python
# Смешанные имена
handle_mixed_script_name("李 Ming")  # -> ('李', 'Ming', '')

# Имена этнических меньшинств
process_ethnic_name("拉木次仁")  # -> ('拉木', '次仁', '')  # Тибетское имя
process_ethnic_name("艾买提·阿不都拉")  # -> ('艾买提', '阿不都拉', '')  # Уйгурское имя
process_ethnic_name("钢巴特尔")  # -> ('钢巴特尔', '', '')  # Монгольское имя

# Конвертация между упрощенными и традиционными иероглифами
simplified_to_traditional("李明")  # -> "李明"  # Остается без изменений, так как эти иероглифы одинаковы
traditional_to_simplified("張三")  # -> "张三"  # Первый иероглиф упрощается

# Редкие фамилии
handle_rare_surname("万俟宗")  # -> ('万俟', '宗', '')
handle_rare_surname("慕容德")  # -> ('慕容', '德', '')

# Нечеткий поиск
fuzzy_match_chinese_name("Li Min", ["Li Ming", "Wang Li", "Zhang Min"])  # -> ("Li Ming", 0.85)
fuzzy_match_chinese_name("Ли Мин", ["李明", "王力", "张敏"], threshold=0.7)  # -> ("李明", 0.82)
```

### 5.4 Пакетная обработка имен
```python
# Пакетная обработка списка имен
names = ["李明", "Wang Xiaohong", "Ли Мин", "欧阳修", "张三"]

# Базовый режим (возвращает кортежи)
result_tuples = batch_process_chinese_names(names)
for name, result in zip(names, result_tuples):
    print(f"{name} -> {result}")

# Расширенный режим (возвращает словари с детальной информацией)
result_dicts = batch_process_chinese_names(names, output_format='dict')
for result in result_dicts:
    print(f"Оригинал: {result['original']}")
    print(f"Фамилия: {result['lastname']}")
    print(f"Имя: {result['firstname']}")
    print(f"Тип: {result['type']}")
```

## 6. Приложения

### 6.1 Список доступных функций
- `is_chinese_char(char)` - проверка, является ли символ китайским иероглифом
- `is_chinese_name(name)` - проверка, является ли строка китайским именем
- `split_chinese_name(name)` - разделение китайского имени на фамилию и имя
- `identify_transliterated_chinese_name(name)` - определение транслитерированного китайского имени
- `transliterate_chinese_name(name, system)` - транслитерация китайского имени в указанной системе
- `compare_transliterations(name, systems)` - сравнение транслитераций имени в разных системах
- `detect_transliteration_system(name)` - определение системы транслитерации имени
- `transliteration_to_hanzi(name, system)` - попытка преобразования транслитерированного имени обратно в иероглифы
- `is_simplified_chinese(text)` - проверка, содержит ли текст упрощенные иероглифы
- `is_traditional_chinese(text)` - проверка, содержит ли текст традиционные иероглифы
- `simplified_to_traditional(text)` - преобразование упрощенных иероглифов в традиционные
- `traditional_to_simplified(text)` - преобразование традиционных иероглифов в упрощенные
- `identify_ethnic_name(name)` - определение этнической принадлежности имени
- `process_ethnic_name(name)` - обработка имен этнических меньшинств
- `handle_rare_surname(name)` - обработка редких китайских фамилий
- `handle_mixed_script_name(name)` - обработка имен со смешанными системами письма
- `fuzzy_match_chinese_name(query, candidates)` - нечеткий поиск китайских имен
- `batch_process_chinese_names(names_list)` - пакетная обработка списка имен
- `detect_japanese_kanji(text)` - определение наличия японских кандзи в тексте
- `calculate_similarity(s1, s2)` - вычисление сходства между двумя строками
- `export_chinese_surnames_to_json(file_path)` - экспорт базы данных фамилий в JSON
- `import_chinese_surnames_from_json(file_path)` - импорт базы данных фамилий из JSON
- `verify_chinese_name_processing(name, expected_result)` - проверка корректности обработки китайского имени
- `manual_correction_interface(name, corrected_parts)` - интерфейс для ручной коррекции разбора имени

### 6.2 Статистика базы данных
- Всего китайских фамилий: более 500
- Покрытие населения Китая: более 99% населения
- Поддерживаемые этнические группы: хань, тибетцы, уйгуры, монголы и другие
- Системы транслитерации: пиньинь, система Палладия, Уэйда-Джайлза, Йельская
